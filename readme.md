# Phemia

`@title: Phemia`  
`@author: tangzihao kongxiaoyi lujiayin`  
`@description: ZJU 2021 Compiler Project`

## Features

1. 这是一个类 C++,Java 的语言编译器（i.e.缝合怪）
2. 预期 features:
   - 支持`int`, `char`, `float`, `double`, `boolean`这些基本数据类型。
   - 删除指针，除了基本数据类型之外均为指针。
   - 删除 `union`, `struct`
   - 支持面向对象，但是不是`C++`的面向对象
   - 支持 forEach 循环
   - 支持矩阵运算
   - 重写垃圾回收机制
